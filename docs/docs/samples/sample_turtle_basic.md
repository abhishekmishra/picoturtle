# turtle_basic

## Program

```lua
-- turtle_basic.lua
-- Author: Abhishek Mishra
-- Date: 15/01/2022
--
-- simplest of turtle files
-- just two commands
--
local t = t or require'picoturtle'.new()
t:forward(100)
t:right(90)
t:forward(100)

```
