# PicoTurtle
A very tiny turtle. Homepage is <a href="https://picoturtle.github.io">here</a>.

### Run from source

You need nodejs v10 or greater.

```
# git clone https://github.com/abhishekmishra/picoturtle.git
# npm install
# npm start
```
